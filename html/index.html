Summary: Latitude Vs. X

This project sought to visualize the data of over 500 cities across the world of varying distance from the equator. I sought to anaylze data and see if there were any realtionships between the weather and its proximity to the equator. In order to accompolish this analysis data of over 500 cities was pulled from the OpenWeather API and a dataset was assembled to begin the process.

After the data was assembled and some inital cleaning was done for example dropping cities whose data wasn't appicable to the data it was being matched against. Then with my cleaner data, I could being plotting using Matplotlib. Various relationships between weather vs. latitude were explored including; temperature, humidity, cloudiness, and wind speed.

This webiste provides source data and visualizations created as part of my analysis, as well as explanations and descriptions of any trends and correlations witnessed.
